# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Enforcement

Instances of unacceptable behavior may be reported by opening a [GitHub Issue](https://github.com/musher-dev/mush/issues).

Project maintainers are responsible for clarifying standards and taking appropriate action in response to reports.

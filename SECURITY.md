# Security Policy

## Reporting a Vulnerability

Please report security vulnerabilities through [GitHub Security Advisories](https://github.com/musher-dev/mush/security/advisories/new).

**Do not** open a public issue for security vulnerabilities.

## Response Timeline

- **Acknowledgment**: Within 48 hours
- **Assessment**: Within 1 week
- **Fix for critical issues**: Within 30 days

## Supported Versions

Only the latest release is supported with security updates.

| Version | Supported |
|---------|-----------|
| Latest  | Yes       |
| Older   | No        |

## Scope

This policy covers the **Mush CLI** only. For issues with the Musher platform API, contact the Musher team directly.

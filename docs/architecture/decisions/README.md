# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) for the Mush CLI.

ADRs capture significant design decisions along with their context and consequences.

## Index

|-----|-------|--------|
| [001](001-scroll-region-rendering.md) | Scroll Region Rendering for Watch UI | Accepted |
| [002](002-worker-tui-governance.md) | Worker TUI Governance | Accepted |

## Adding a New ADR

1. Copy the next available number: `NNN-short-title.md`
2. Fill in the header fields (Status, Date, Context, Decision)
3. Write up the context, options considered, and rationale
4. Add an entry to the index table above
5. Submit as part of your PR
